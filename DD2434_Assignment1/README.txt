USAGE:
- Plots can be generated by running the main.py program, no arguments required.
- Each run will only use one of the three algorithms (PCA, MDS, isomap)
- To change algorithm, there is a hardcoded variable on line 17 of main.py. It is a string that should have one of the following contents:
"PCA", "MDS", or "isomap"

DEPENDENCIES:
- Python 3.5 or higher
- Numpy
- Matplotlib
